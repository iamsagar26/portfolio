# 🚀 Sagar Jadhav — Personal Portfolio

A modern, single-page developer portfolio built with pure HTML, CSS, and vanilla JavaScript. Designed with a clean editorial aesthetic, smooth animations, and polished UI interactions.

---

## ✨ Features

- **Custom animated cursor** with mix-blend-mode effect and hover scaling
- **Bento-grid hero section** with illustrated cards
- **Smooth scroll animations** via IntersectionObserver (reveal on scroll)
- **3D tilt effect** on project cards
- **Parallax scrolling** on select elements
- **Auto-hiding navbar** that reappears on scroll-up with active section highlighting
- **Logo ticker / marquee** for tech stack display
- **FAQ accordion** section
- **Floating "Hire me" CTA** that smooth-scrolls to the contact section
- **Fully responsive** design for mobile and desktop
- **No frameworks, no build tools** — pure HTML/CSS/JS, zero dependencies

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Markup | HTML5 |
| Styling | CSS3 (custom properties, grid, flexbox, animations) |
| Scripting | Vanilla JavaScript (ES6+) |
| Fonts | Google Fonts — DM Sans, Playfair Display, Caveat |

---

## 📁 Project Structure

```
portfolio/
└── index.html        # Single self-contained file (HTML + CSS + JS)
```

Everything lives in one file — no external assets, stylesheets, or scripts required beyond Google Fonts.

## 🚀 Getting Started

No installation or build step needed.

```bash
# Clone the repo
git clone https://github.com/iamsagar26/portfolio.git

# Open in browser
open index.html
```

Or just drag and drop `index.html` into any browser.
